# react-lightweight-boiler
A lightweight boilerplate featuring React, Redux, Webpack, Sass.

## Installation
```bash
npm install
```

## Running Dev Server
```bash
npm start
```
## Viewing the application
Visit [http://localhost:5000](http://localhost:5000) to view the application
